# Decision Log

Use decision logs for architecture and library choices with long-term impact.

## File naming

- Format: `NNN-short-title.md`
- Example: `001-cloudflare-workers-and-supabase.md`

## Template

Each decision file should contain:

- Date
- Status (`Proposed`, `Accepted`, `Superseded`)
- Context
- Decision
- Consequences
