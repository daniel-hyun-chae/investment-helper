# Backlog-Planner Override

## Vocabulary

Use these terms consistently in all backlog items, specs, and conversations:

| Term | Meaning |
|------|---------|
| watchlist | A set of company identifiers a user subscribes to for updates. |
| disclosure | A public filing or market-relevant announcement from a source API. |
| filing event | A normalized event derived from one disclosure payload. |
| connector | A source-specific adapter that fetches and normalizes external API data. |
| subscription | A durable user preference linking a user to a company/source pair. |
| analysis job | An async task that transforms normalized data into investment insights. |
